Canvas is a game where you draw pixels on a canvas and create things.
Please:
  • Be nice. Don't cause chaos or destroy things people have obviously worked hard on.
  • Drawings should be SFW. Kids use this.
  • Don't hack or use bots.
Code is on Github. Keyboard controls:
  • Arrow keys / WASD - move
  • Z/X - Zoom
  • C - toggle placing
Scroll to zoom, click and drag to move. Everything should just work normally on mobile.
